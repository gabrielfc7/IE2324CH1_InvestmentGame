Welcome

Build a simple stock portfolio with a $100 budget and track performance. Modern dark UI with TTU highlights.

Quick Start

  1. Open Portfolio, set Name, Section, and Semester.
  2. Use +/- to buy whole shares. Stay at or under $100 and max 5 stocks.
  3. Submit. Data is recorded to the class sheet via Apps Script.

Your anonymous uid is stored locally (no email needed).
